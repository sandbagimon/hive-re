# SimLab Iteration Log

这个目录用于记录 SimLab 每次迭代的详细内容。每个迭代文件应说明本轮目标、主要改动、验证结果、已知限制和下一步建议。

## 记录规范

新迭代文件命名格式：

```text
YYYY-MM-DD-短标题.md
```

建议内容结构：

```text
# 迭代标题

日期：
提交：

## 目标
## 主要改动
## 验证
## 已知限制
## 下一步
```
